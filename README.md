###Corporate Credit Risk Assessment Using Financial Statement Data

This project analyzes credit risk using financial statement data. Due to the volume of data, all raw datasets are included in the uploaded ZIP archive.

##Project Contents
Company Folders: Organized data for each company, including BalanceSheet.csv, CashFlow.csv, and IncomeStatement.csv.
Python Scripts: Code for Data Cleaning, Regression Analysis (CreditRiskAssessment.py), and the interactive dashboard (app.py).
Processed Data: The FINAL_OUTPUT.xlsx file containing calculated ratios and risk scores.
Documentation: Final project report in Word format.

##Installation and Usage
Dependencies: Install the required libraries using the command: pip install -r requirements.txt
Analysis: Run CreditRiskAssessment.py to perform data cleaning and model training.
Dashboard: Launch the interactive interface by running: streamlit run app.py

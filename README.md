# Corporate-Credit-Risk-Assessment-Using-Financial-Statement-Data
This project analyzes credit risk using financial statement data. Due to the volume of data, all files are included in the uploaded ZIP archive.

📂 What's Inside:
Company Folders: Organized data for each company, including BalanceSheet.csv, CashFlow.csv, and IncomeStatement.csv.
Python Scripts: Code for Data Cleaning, Regression Analysis, and a Streamlit dashboard.
Documentation: Final project report in Word format.
